一根羊肉串撸出千亿元市场
发布时间：2017-02-28 浏览人数：5

过去一提到羊肉串，大家都联想到路边摊，而且容易让消费者担心食品安全问题。而且，由于羊肉串标准化能力偏弱，造成国内出名的羊肉串连锁品牌并不多。然而，羊肉串的市场却并不小，甚至比鸭脖市场大。

一根羊肉串撸出千亿元市场

一家不是很出名的专门卖羊肉串的餐饮企业阿拉提羊肉串大王荣获了千万元融资，这背后的羊肉串市场有多大？融资千万，估值过亿！一根羊肉串也能撸出千亿元市场！单品店在新的2017年将持续在餐饮行业里刷新大家对它的认识。一家不是很出名的专门卖羊肉串的餐饮企业阿拉提羊肉串大王荣获了千万元融资，让一起来看看传统的羊肉串市场是如何吸引资金的注意力。

羊肉串如何突破？

2月21日消息，阿拉提餐饮集团宣布获得首轮千万元融资，投资方为顺融资本，估值超过一个亿。旗下连锁品牌“阿拉提羊肉串大王”在2017年将重点发展。“阿拉提羊肉串大王”是阿拉提餐饮集团旗下连锁餐饮品牌之一，目前在苏州各大购物中心均有覆盖。

并已在杭州、上海、北京、常州、无锡等城市开设近50家直营店，单店面积最小仅十几平米，目前已成长为国内最知名的羊肉串连锁品牌，深受消费者喜爱。预计2017年将新开直营店、加盟店各100家。此外，还将扩大产能，对原料基地进行建设，加强研发等。阿拉提餐饮集团创始人兼董事长樊荣，曾担任小南国餐饮集团市场及拓展总监、华东区总经理，吴地人家餐饮集团总裁。多年从事餐饮业的经历，让他对餐饮业的发展有自己的看法。

目前市面上烤串的产品在生产和销售时候存在着以下问题：

一、消费场景的局限。

过去一提到羊肉串，大家都联想到路边摊，而且容易让消费者担心食品安全问题。

二、标准化问题。

餐饮的标准化被认为是实现餐饮品牌持续生命力和竞争力很重要的方面，但是市面上真正做好这一点的并不多。从两方面的思路出发，在餐饮产品的标准化上，阿拉提专门成立了全国首家专业烧烤研发机构“阿拉提羊肉串研究院”。团队聘请多名资深烧烤大师，对包括原料、腌料、串法、烤法、蘸料、竹签及烤炉等各个环节进行研发。除了这些环节把关之外，为从根本上解决因产能及配送不足引起的发展瓶颈，保证在全国范围的强大销售能力。

“阿拉提”设立了羊肉串中央厨房（一期），配以**冷库，专业的冷链物流配送团队，满足苏州、杭州以及常州等长三角城市百家门店的供货需求，这也为进一步拓展长三角乃至全国市场打下了基础。

一根羊肉串的商业模式

在改变用餐场景上，阿拉提摒弃以往烧烤类食品沿街开店的传统模式，在羊肉串研究院的支撑下，实现了烤串去烟化，从而顺利进驻了各大地标商业中心等主流人群集中地，引起了年轻消费者的关注和追捧。同时阿拉提针对不同消费者需求，开通了多样化的售卖渠道，包括堂吃、外带、外卖，全天候不间断售卖。在增加流量上，阿拉提已与大众点评、美团等流量平台进行合作，为外卖平台提供专门取餐口，大大提高了平效。

商业模式上，阿拉提希望能够从供应链入手，打造一个产业链的闭环。目前，在供应链端，阿拉提选用的羊肉都是来自腾格里沙漠南缘的“娜尔科”牧场，纯净弱碱性有机甘草羊，是中草药甘草叶全程喂养，除了天然鲜美的肉质，更强化了羊肉本身的多种营养价值。未来，阿拉提还希望打造半成品电商。

比如，向需要在家庭场景进行烧烤的C端消费者，出售专业的烤炉和烧烤工具和标准化的半成品肉串。依托线下的门店优势，向线上的平台扩展。樊荣告诉我们，此轮融资将主要用于：线下店铺扩张和品牌推广；扩大产能，同时对原料基地进行建设，用于产出更高品质的产品；加强研发，引进一些信息化系统进行管理。

羊肉串市场比鸭脖还大？

此次投资方负责人，顺融资本副总裁张明也简述了这个项目的投资逻辑：

1、羊肉串国内的市场在千亿元级别，比鸭脖还大。

鸭脖在国内每年消费在八百亿元，已经涌现出周黑鸭、绝味鸭脖等大型连锁企业。顺融方面相信，中国传统的羊肉串，也可以涌现出品牌化的连锁巨头。

2、团队均来自大型餐饮企业，有丰富管理经验和品牌运作经验。

3、消费场景的改变

阿拉提通过食品安全的严苛保障，加强店面卫生的管控，并把店面从路边搬到主流的商超内，旨在打造一个安全卫生高端的主流餐饮品牌。

4、平效高，盈利能力强

主要做商场内的外带而不是堂食，平均店面控制在二十平米以内，每家店面只需配备2-3人，管理也统一实现标准化。公司业务的毛利率较高，该项目从一开始启动，就实现了盈利。张明也透露，此次融资将帮助公司迅速扩展直营店规模，让阿拉提品牌迅速拓展到全国市场。据悉，阿拉提计划未来3年门店数超千家。
